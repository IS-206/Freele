Freele

The Open Source chat-client

We want to create a chat-program suitable for a small amout of users. You can host your own server and you just need to send the IP-address and port to the other users. The messages are encrypted so you can chat safe. You can chat in groupchat and privatechat

The program is written in java since this is the programming language we are most common with. Our focus is to get contributers to the project so that everyone can share opinions and experiences.

The project is Open Source and everyone can join. Just Fork the project and send in a pull request!
